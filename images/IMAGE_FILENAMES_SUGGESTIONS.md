## 実装方法

これらの画像ファイルを `web/images/` フォルダに配置した後、`reader.js` の `backgroundKeywords` オブジェクトに以下を追加してください：

```javascript
// 事件・イベント背景
'images/event_murder_ashtray.jpg': ['重厚なガラス製の灰皿を取り出した', '梨奈が灰皿を振り上げる', '鈍い音が響いた'],
'images/event_battery_fire.jpg': ['UPS（無停電電源装置）のパネルを開けた', 'リミッター、切ります', '猛烈な白煙が噴き出した'],
'images/event_burning_embrace.jpg': ['爆発的な赤い閃光が視界を埋め尽くす', '炎の中で、私は梨奈を抱きしめた', '骨まで溶かす熱風'],
'images/event_forged_document_closeup.jpg': ['契約書の偽造を命じられた', '震える手で、架空の署名をなぞった', '偽造した契約書がある'],
'images/event_audit_tension.jpg': ['内部監査室の人間がフロアを歩いている', '会計監査が始まった', '監査の緊張'],
'images/event_toilet_intimacy.jpg': ['トイレの個室に鍵を掛けて二人きり', '狭い密室で向き合う'],
```

## 画像作成のヒント

### event_murder_ashtray.jpg
- 重厚なガラス製の灰皿をクローズアップ
- 暗い背景、緊迫感のある構図
- 血痕や暴力性を示唆する要素

### event_battery_fire.jpg
- サーバーラックと発火中のUPS
- 白煙や火花が見える
- 赤やオレンジの発光

### event_burning_embrace.jpg
- 炎に包まれる二人のシルエット
- 赤とオレンジの炎のテクスチャ
- 美しくも破滅的な雰囲気

### event_forged_document_closeup.jpg
- 契約書の用紙をクローズアップ
- ペンで署名する手元
- 緊張感のある照明

### event_audit_tension.jpg
- オフィスの会議室や廊下
- 監査書類やファイル
- 緊張した雰囲気の色調

### event_toilet_intimacy.jpg
- トイレの個室内の狭い空間
- 二人が接近している構図
- 密室感と緊張感
