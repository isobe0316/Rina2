# 美咲の箱 - 新曲「盾にならない箱」

## コンセプト
- 静かで内省的
- ピアノとストリングス中心
- 謎めいた雰囲気
- 少し切なさを含む
- 過去の恋人（美咲）との記憶と、今の複雑な感情を表現

## Suno用 歌詞

```
[Verse 1]
透明な箱の中
君の笑顔が揺れている
触れられない記憶
ガラス越しの温もり

[Pre-Chorus]
守りたかったもの
守れなかったもの
箱は何も語らず
ただ、そこにある

[Chorus]
盾にならない箱
割れもしない距離
君はもういないのに
いつまでも見ている
すりガラスの向こう
声も届かないまま
残された影だけが
今日も揺れている

[Verse 2]
数字の隙間に
隠した小さな嘘
誰かのためだった
誰も救えなかった

[Pre-Chorus]
正しさを選べば
君は遠ざかる
でも、このまま
箱に閉じたままで

[Chorus]
盾にならない箱

君はもういないのに
いつまでも見ている
すりガラスの向こう
声も届かないまま

[Bridge]
喫茶店の窓越しに
誰かの横顔見たとき
君じゃない誰かが

胸の奥の箱が

[Final Chorus]

開けることもできず
```

## 音楽的ディレクション

### テンポとキー
- **BPM**: 60-70（ゆっくり、静か）
- **キー**: Cmajor または Aminor（物悲しい雰囲気）

### 楽器構成
1. **ピアノ**: メインメロディと伴奏
   - シンプルなアルペジオ
   - 余白を大切に
   
2. **ストリングス**: 
   - バイオリンとチェロ
   - 控えめなレイヤー
   - サビで少し盛り上げる

3. **その他**:
   - 最小限のパーカッション（または無し）
   - 環境音（オプション：雨音、風など）

### ボーカル
- 男性ボーカル
- 感情を込めつつ、抑制された表現
- 囁くような箇所と、少し感情が溢れる箇所のコントラスト

### ダイナミクス
- 全体的に静か（p〜mp）
- サビで少しだけ強く（mf）
- アウトロで徐々にフェードアウト
