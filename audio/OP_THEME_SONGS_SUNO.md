## 使用方法

### Sunoでの生成手順

1. **各曲を個別に生成**
   - 上記の「音楽スタイル」をStyle欄にコピー
   - 「歌詞」をLyrics欄にコピー
   - Instrumental: OFF（ボーカル入り）
   - Generate

2. **生成後の処理**
   - 各mp3ファイルを `web/audio/` ディレクトリに配置
   - ファイル名: `op_theme_1.mp3`, `op_theme_2.mp3`, `op_theme_3.mp3`

3. **タイトル画面で自動ランダム再生**
   - `app.js`が自動的にop1, op2, op3からランダムに選択
   - ユーザーがタイトル画面を開くたびに異なる曲が流れる

## 技術的な実装

### audio.jsでの定義
```javascript
this.tracks = {
    op1: 'audio/op_theme_1.mp3',
    op2: 'audio/op_theme_2.mp3',
    op3: 'audio/op_theme_3.mp3',
    // ...
};
```

### ランダム再生ロジック
```javascript
if (trackName === 'op') {
    const opTracks = ['op1', 'op2', 'op3'];
    trackName = opTracks[Math.floor(Math.random() * opTracks.length)];
}
```

タイトル画面で `audioManager.play('op')` を呼び出すと、自動的に3曲の中からランダムに1曲が選ばれて再生されます。
